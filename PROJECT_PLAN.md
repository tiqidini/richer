# План проекта: Хроники Джека Ричера

## Обзор
Веб-приложение для отслеживания прогресса чтения серии книг о Джеке Ричере автора Ли Чайлда.

## Функции
- Список всех книг о Джеке Ричере.
- Отметка книг как прочитанных/непрочитанных.
- Индикатор прогресса чтения.
- Детальный вид для каждой книги с описанием и персонажами.
- Современный интерфейс с использованием Glassmorphism и анимаций.

## Архитектура
- **Фронтенд**: React, TypeScript, Vite.
- **Стилизация**: Tailwind CSS.
- **Управление состоянием**: React Hooks (useProgress).
- **Данные**: Статический файл JSON/TypeScript (`books.ts`).
